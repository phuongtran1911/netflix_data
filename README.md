# netflix_data
**Investigating Netflix Movies**

**Netflix**! Given the large number of movies and series available on the platform with the dataset `netflix_data.csv`, along with the following table detailing the column names and descriptions, exploratory data analysis on the `netflix_data.csv` data to understand more about movies from the 2000s decade.

## The data
### **netflix_data.csv**
| Column | Description |
| ------ | ----------- |
| `show_id` | The ID of the show |
| `type` | Type of show |
| `title` | Title of the show |
| `director` | Director of the show |
| `cast` | Cast of the show |
| `country` | Country of origin |
| `date_added` | Date added to Netflix |
| `release_year` | Year of Netflix release |
| `rating` | Rating of the show |
| `duration` | Duration of the show |
| `listed_in` | Show genres |
| `description` | Description of the show |
